# **说清楚LLaMA、GPT、BERT的核心区别**
这三个模型的区别主要在于架构和训练任务。

GPT 用的是只有 Decoder 的结构，注意力是因果掩码，也就是每个 token 只能看到它左边的 token，训练任务是预测下一个 token，所以它特别擅长生成文本、做对话。

BERT 只有 Encoder，注意力是双向的，每个 token 可以看到整个序列。预训练任务包括掩码语言模型和下一句预测，所以它更适合理解类的任务，比如文本分类、问答、实体识别。

LLaMA 虽然也是只有 Decoder、因果掩码、自回归生成，但它在具体组件上做了很多改进，比如用了 Pre‑RMSNorm、SwiGLU 激活函数、旋转位置编码 RoPE。它主要是作为一个开源的基础模型，既可以做生成，也可以微调后做理解。
